# ml-experiment-playground
> 🧪 A personal sandbox for experimenting with machine learning frameworks, libraries, and tools.

## About

This repository is my personal playground for exploring and learning new ML/AI technologies. It's a space for hands-on experimentation without the constraints of production code—expect work-in-progress scripts, rough prototypes, and learning exercises.

**This is not production code.** It's meant for learning, tinkering, and having fun with new tools.

## Current Experiments

### MLflow Experiments (`mlflow-experiments/`)

Exploring [MLflow](https://mlflow.org/) for experiment tracking, model logging, and reproducibility:

#### Tech Stack

- **ML Frameworks:** PyTorch, scikit-learn
- **Experiment Tracking:** MLflow
- **Hyperparameter Tuning:** Optuna
- **Environment Management:** Pipenv

#### Getting Started

```bash
cd mlflow-experiments
pipenv install
pipenv shell
```

Then run any experiment:

```bash
python sklearn_logging_exp.py
```
